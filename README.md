# odin-recipes
The recipe project for The Odin Project

The purpose of this project is to solidify my HTML skills by creating a basic
website to host recipes. The site will have a homepage that links to various
recipe pages, complete with images.

By the end of this project I should have a strong grasp of:

	- Building HTML boilerplates.
	- Using basic elements to make paragraphs and headings.
	- Making text bold/strong or italicized/emphasized.
	- Using ordered and unordered lists.
	- Adding links and images.
	- Using a basic git workflow to update my work.
